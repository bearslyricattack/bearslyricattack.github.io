# 课程

马克思主义原理

山东·威海

马克思主义原理

1.世界的物质性及发展规律:

1.1 哲学:

哲学：系统化、理论化的世界观，是对自然知识、

社会知识和思维知识的概括和总结，它提供了对于世界以及人与世界关系的全面而深刻的思考。

全部哲学，特别是近代哲学的重大的基本问题，是思维和存在的关系问题。

1.2 唯物主义和唯心主义

主观唯心主义:主体之内的精神是世界的本原，认为世界是人的主观意志的产物或表现。

客观唯心主义:主体之外的精神是世界的本原，认为世界是客观精神的产物或表现。

朴素唯物主义:根本观点：用某一种或几种具体的“原初”的物质形态解释世界本原，如水、火、气等，并认为物质形态是运动变化的。

形而上学唯物主义:原子是组成物质的最小单位.

辩证唯物主义:物质.物质的存在:运动,物质决定意识.

人工智能是人类智能的物质化.

1.3 同一性:

思维和存在有无同一性，即思维能否认识存在，能否正确认识存在的问题。

唯物论者、彻底唯心论者是可知论者.

1.4 辩证法:

事物的联系,发展和矛盾.

矛盾的同一性和斗争性.普遍性和特殊性.

量变与质变.

辩证的否定观念:扬弃.既克服又保留.

1.5 实践与认识:

实践是认识的来源.是认识发展的动力.是认识的目的.是检验真理的唯一标准.

认识的本质是主体在实践基础上对客体的能动反映.

1.6 真理:

客观性,绝对性,相对性.

真理的内容表明了主客观统一的确定性和发展的无限性.

人们在一定条件下对事物的客观过程及其发展规律的正确认识总是有限度的。

实践是检验真理的唯一标准.

实践标准有确定性和不确定性.

2. 人类社会发展规律:

历史观: 人类对社会历史发展的根本看法和根本观点.

社会存在决定社会意识.

社会意识对社会存在具有能动的反作用.

2.1 生产力与生产关系:

生产力决定生产关系.

生产关系要适合生产力.

2.2 经济基础与上层建筑:

经济基础决定上层建筑.

上层建筑对经济基础具有能动的反作用.

2.3 社会形态更替:

社会形态:经济基础+上层建筑.

根本属性: 生产资料所有关系.

统一性与多样性: 社会演进情况统一.具体过程多样.

必然性,历史选择性.

前进性,曲折性,顺序性,跨越性.

2.4 社会历史发展的动力:

根本动力:社会基本矛盾:即生产力和生产关系的矛盾，经济基础和上层建筑的矛盾.

直接动力:阶级斗争和社会革命.

2.5 人民群众在历史发展中的作用:

人民群众是历史的创造者.

辩证的评价英雄领袖的作用.

3. 资本主义的本质及规律:

商品经济:交换.

商品:使用价值和价值.

具体劳动和抽象劳动(脑力).

3.1 价值规律:

商品价值量是由生产商品的劳动时间决定的.

价格围绕价值上下波动.

3.2 私有制的基本矛盾:

私人劳动和社会劳动的矛盾构成私有制商品经济的基本矛盾。

矛盾产生基础：社会分工和私有制。

矛盾解决途径：交换。

3.3 马克思劳动价值论:

资本主义经济制度是以资本主义私有制和雇佣劳动为基础的一种剥削制度.

3.4 剩余价值:

劳动不是商品，没有价值.

剩余价值转换为利润.

劳动二重性,一句话总结：具体劳动创造使用价值，抽象劳动创造价值。

4. 资本主义的发展及其规律:

垄断资本主义.
